# PhysicsGame

Repo ini berisi catatan pembelajaran tentang `pygame`, game logic, dan physics sederhana.

## Daftar Isi Pembelajaran

### 1. Bouncing Ball

Yang dipelajari:

- dasar game loop `pygame`
- penggunaan `delta time` (`dt`)
- gravitasi pada sumbu `y`
- hubungan percepatan, kecepatan, dan posisi
- collision bola dengan lantai
- efek bounce atau pantulan
- damping atau pengurangan pantulan
- perbedaan pendekatan `v1` dan `v2`
- kenapa `v2` lebih rapi dan lebih baik secara logic

Dokumentasi detail:

- `documentasi/analisis-bouncing-ball.md`

## Topik Berikutnya

Topik ini bisa ditambah seiring pembelajaran berikutnya:

- jump mechanic
- wall collision
- multiple bouncing objects
- friction
- projectile motion
- simple platformer physics
- collision antar object

## Catatan

`README.md` ini dipakai sebagai daftar isi pembelajaran.

Setiap topik sebaiknya punya dokumentasi terpisah di folder `documentasi/`.
